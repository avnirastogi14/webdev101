Images and Tables

Attributes in the image tag:

  • src: Specifies the path to the image file.
  • alt: Provides alternative text for the image, which is displayed if the image cannot be loaded.
  • width: Sets the width of the image in pixels or percentage.
  • height: Sets the height of the image in pixels or percentage.
  • title: Provides additional information about the image, often displayed as a tooltip when the mouse hovers over the image.
  • loading: Specifies how the browser should load the image (e.g., "lazy" for lazy loading).
