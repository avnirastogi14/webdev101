Tables

Tables are used to display data in a structured format. They consist of rows and columns, where each cell can contain text, numbers, or other elements.

In HTML, tables are created using the <table> element, with rows defined by the <tr> element and cells defined by the <td> element for data cells or <th> for header cells.

Here's a simple example of a table:

Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3

In this example, the first row contains headers, while the subsequent rows contain data cells. The border attribute is used to add a border around the table and its cells.

Tables can also include attributes like colspan and rowspan to merge cells across columns or rows, respectively. For example:

Merged Header Header 3
Merged Cell Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 2 Row 2, Cell 3

In this example, the first header spans two columns, and the first cell in the second row spans two rows.

Tables can be styled using CSS to improve their appearance, such as adding padding, changing background colors, or adjusting borders.
